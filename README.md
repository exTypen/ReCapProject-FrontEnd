<h1 align="center"> Recap Frontend  </h1>

## About

This project is the frontend part of <a href="https://github.com/cenkerkumlucali/ReCapProject" target="_blank" rel="nofollow"> ReCapProject-backend</a> 

### Project Content
- CRUD Operation
  - Add
  - Read
  - Update
  - Delete
- Filter
  - Multiple filtering
- Auth
  - Login
  - Register
  - Profile Update
  - Password Update
- Reactive Forms
- Validation processes
  - Toastr
- Data transfer between Components, Child to Parent Data Binding
- Locale Storage control
- Models
- Services
- Custom pipes
- Interceptor
- Bootsrat v5 Beta 2
- Bootsrap Icons 
- Jquery 3.6.0
- ngx-toastr 13.2.1
- Popper.js 1.16.1
- Primeicons 4.1.0
- Primeng 11.3.1
